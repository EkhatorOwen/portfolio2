Collaborate

This project was built for teams to help them collaborate. In this app, a team lead can sign up, log in, create projects, add team members to the project. Each project created has 4 features; a 'Schedule' page that allows team leads to add calendar items, it also syncs with google calendar, a 'Manage Task' page that allows team leads to create and assign task to team members, a 'Visuals' page that shows the current status of task completed vs task pending, a 'Message Board' that allows project members to add, delete & edit messages. The app also has a chat functionality that allows team leads and team members to chat. There is an option to alert a team member by SMS when they are added to a project but that is currently only available in test and not on production.

Technologies: React, Redux, Express, Postgres, SendGrid(for emails), Pusher(for chat messaging), firebase(for storage), Twilio(for text messaging), Froala Editor, ChartJS for visualization.
